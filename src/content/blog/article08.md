# 线性代数

## 矩阵

### 基本运算

| -      | 加法 | 乘法 |
| ------ | ---- | ---- |
| 分配律 | ✅   | ✅   |
| 结合律 | ✅   | ✅   |
| 交换律 | ✅   | ❌   |

1. $A^{-1} = A 元素取倒数$
2. A != |A|
3. 主对角$\begin{bmatrix} A & O \\ O & B \end{bmatrix}^{n} = \begin{bmatrix} A^n & O \\ O & B^n \end{bmatrix}$
4. 副对角\*(对调)$\begin{bmatrix} O & A \\ B & O \end{bmatrix}^{n} = \begin{bmatrix} O & B^n  \\ A^n & O \end{bmatrix}$
5. 求$A^n = PB(对角矩阵) P^{-1} = PB^{n} P^{-1}$
6. $(A+B)^T = A^T + B^T$ (只有转置可以)
7. $AA^T != A^TA$

### 矩阵的逆

1. $|A^{-1}| = |A|^{-1}$
2. 初等变换不改变矩阵的秩 $r(A) = r(PA) = r(AQ), $P, Q 分别是 m、n阶可莉矩阵

### 矩阵的秩

1. 秩相同 $r(A) = r(A^T) = r(AA^T) = r(A行) = r(A列)$
2. $r(A^{*}) = n, r(A) = 1$
3. $r(A^{\*}) = 1, r(A) = n -1 $
4. $r(A^*) = 0, r(A) < n - 1$
5. $r(A + B) \leq r(A) + r(B)$
6. $r(AB) \leq min(r(A), r(B))$
7. $AB = O, r(A) + r(B) \leq n$
8. 非零矩阵的秩r≥1
9. $齐次A_{m\times{n}}X=0, 解集的秩r = A列数 - r(A)$
10. $非齐次A_{m\times{n}}X=b, 解集的秩r = A列数 - r(A) + 1$

### 伴随矩阵

1. $AA^* = A^*A = |A|E$
2. $|A^*| = |A|^{n-1}$
3. $A^* = |A|A^{-1}$
4. $(kA)^* = k^{n-1}A^*$
5. 二阶矩阵求伴随 = `主对角线对调 副对角线变号`

### 初等变换和初等矩阵

#### 初等变换

1. 数乘（非0） 行/列
2. 交换某两行/列
3. 某行/列 k倍 加到 另一行/列

#### 性质和公式

1. $E^T = E$
2. $E^{-1} = E$
3. $E_{倍乘k}^{-1} = E_{倍乘\tfrac{1}{k}} (取倒数)$
4. $E_{倍加k}^{-1} = E_{倍加 -k} (取相反数)$

### 等价矩阵

1. 同形 r(A) = r(B)

## 线性方程组

齐次 有解的条件
$r(A) = n,（向量组线性无关） 方程组有唯一0解$
$r(A) < n,（列向量组线性相关） 方程组有n-r个线性无关解$

非齐次 有解的条件
$r(A) != r([A, b]),（b不能由线性相关的向量组线性表出）， 则无解$
$r(A) = r([A, b]) = n,（向量组线性无关，增广线性相关），有唯一解$
$r(A) = r([A, b]) = r < n, 有无穷多解$

抽象性 有解的条件
$Ax=0: 总有解，至少有0解$
$A_{m \times n}x=0: r(A) = n，只有0解；r(A) < n，有无穷多解$
$A_{m \times n}x=b: r(A) != r([A, b])，无解；r(A) = r([A, b]) = n，有唯一解；r(A) = r([A, b]) = r < n，有无穷多解$

## 向量组

## 特征值和特征向量

| 矩阵     | $A$       | $A^2+E$           | $A^{-1}$              | $f(A)$       | $A^*$                    | $A^T$      |
| -------- | --------- | ----------------- | --------------------- | ------------ | ------------------------ | ---------- |
| 特征值   | $\lambda$ | $\lambda{}^2 + 1$ | $\tfrac{1}{\lambda} $ | $f(\lambda)$ | $\tfrac{\|A\|}{\lambda}$ | $\lambda$  |
| 特征向量 | -         | -                 | -                     | -            | -                        | 与其他不同 |

### 特征值性质

1. 特征值的和 = tr(A)
2. 特征值的乘积 = |A|

### 矩阵的相似

存在可莉矩阵 P 使得 $P^{-1}AP = B 则 A相似于B$，A~B

1. r(A) = r(B)
2. |A| = |B|
3. $|\lambda{E} - A| = |\lambda{E} - B|$
4. A、B有相同的特征值
5. $A^m$~$B^m$, $f(A)$ ~ $f(B)$, $A^T$~$B^T$, $A^*$~$B^*$，
6. A + kE ~ B + kE

### 相似对角化

1. n阶矩阵A可对角化 <=> 有n个线性无关的特征向量
2. n阶矩阵A可对角化 <=> 任意k重特征根有k个线性无关的特征向量
3. 对称矩阵 => 可对角化
4. 有n个互不相同的特征值 => 可对角化

### 实对称矩阵

1. 不同特征值对应的特征向量相互正交
2. 对角线元素为特征值

## 二次型

### 标准形

$\Sigma_{i=1}^n 特征值_i \times y_i^2$

### 惯性定理

1. 正负惯性指数 = 特征值的正负个数
2. 二次型的秩r = p + q (正负惯性指数之和)

### 合同

1. r(A) = r(B)
2. A、B 有相同正负惯性指数

### 正定

1. 正惯性指数 p=n
2. A 合同 E
3. 顺序主子式大于0
