## 2 进程与线程

### 易错帽子题

- 系统线程 可重复使用
- `中断向量`是`中断服务程序`的`入口地址`
- 互斥信号量 初始值为1
- int类型变量，读写时需要 互斥信号量保护更新

### 用户级、内核级线程

| -    | 用户级线程                                       | 内核级线程             |
| ---- | ------------------------------------------------ | ---------------------- |
| 切换 | 同一进程内不需要内核调度，跨进程需要内核参与调度 | 需要从用户态转到核心态 |
| -    | 由线程库管理                                     | 内核空间管理           |
| 阻塞 | 一个线程阻塞 整体阻塞                            | -                      |

### 进程通信

1. 共享存储 （可直接访问的共享空间）
2. 消息传递 （全双工）
3. 管道 （半双工）

### C语言 分配

1. 动态分配的内容 -> 堆
2. 传递参数和局部变量 -> 栈
3. 常量 -> 正文段

### 处理机调度

1. `进程调度`是操作系统必须的
2. 高级调度（作业调度）外存 -> 内存
3. 中级调度（内存调度）内存 -> （暂时不能运行）外存 -> （满足条件后）内存
4. 低级调度（进程调度）就绪队列 -> CPU

#### 调度指标

1. 周转时间 = 作业完成时间 - 作业提交时间
2. 平均周转时间 = ($\Sigma_{i=1}^n 作业_i的周转时间$)/n

#### 调度算法

注意事项：

1. 区分抢占/非抢占式
2. 区分 临界区、内核临界区（不能处理机调度）

##### 优先级调度算法

优先级的分类

1. 动态优先级
2. 静态优先级

优先级参照原则

1. I/O型进程 > 计算型进程
2. 交互型进程 > 非交互型进程

##### 高响应比优先调度算法

依次选择响应比最高的进行调度
响应比 = $\dfrac{要求服务时间 + 等待时间}{要求服务时间}$
