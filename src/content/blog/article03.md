# 2 进程与线程

系统线程 可重复使用

## 用户级、内核级线程

| -    | 用户级线程                                       | 内核级线程             |
| ---- | ------------------------------------------------ | ---------------------- |
| 切换 | 同一进程内不需要内核调度，跨进程需要内核参与调度 | 需要从用户态转到核心态 |
| -    | 由线程库管理                                     | 内核空间管理           |
| 阻塞 | 一个线程阻塞 整体阻塞                            | -                      |

## 进程通信

1. 共享存储 （可直接访问的共享空间）
2. 消息传递 （全双工）
3. 管道 （半双工）

## C语言 分配

1. 动态分配的内容 -> 堆
2. 传递参数和局部变量 -> 栈
3. 常量 -> 正文段
