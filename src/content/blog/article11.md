# 数据的表示和运算

## 定点数的编码表示

### 原码 补码 反码 移码

> 0的补码表示唯一
> 补码定点数运算时 符号可以作为数的一部分参加运算
> 32位补码表示整数范围$-2^{31} ~ 2^{31} - 1$

| -    | 原码 (首位符号位) | 补码     | 反码                | 移码 |
| ---- | ----------------- | -------- | ------------------- | ---- |
| 正数 | 1                 | 同原码   | 同原码              | -    |
| 负数 | 0                 | 反码 + 1 | 符号位不变 其他取反 | -    |

## 运算方法和运算电路

### 补码定点数加减运算溢出判别

- 一位符号位
  $V= A_s$ && $B_s$ && $S_s $ || $!A_s$ && $!B_s$ && $!S_s$, V = 1溢出
- 双符号位 $S_1 S_2$
  1.  $S_1 S_2 = 00$ 正数
  2.  $S_1 S_2 = 01$ 正溢出
  3.  $S_1 S_2 = 10$ 负数溢出
  4.  $S_1 S_2 = 11$ 负数
- 一位符号根据数据进位情况判断溢出 $V = C_s 异或 C_1$, V=1溢出

### C语言中的整数类型及类型转换

有符号无符号之间转换

- 有符号数用补码
- 无符号数用二进制

不同字长之间转换

- 大字长->小字长 高位截断
- 小字长->大字长 高位用符号拓展（无符号0拓展）

### 数据的存储和排列

#### 大小端存储

```c
  int i = 01 23 45 67H;
```

| -        | 0800H | 0801H | 0802H | 0803H |
| -------- | ----- | ----- | ----- | ----- |
| 大端对齐 | 01H   | 23H   | 45H   | 67H   |
| 小端对齐 | 67H   | 45H   | 23H   | 01H   |
